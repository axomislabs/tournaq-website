Advanced User Administration

User administration is part of the long-term platform vision. These features target established clubs, federations, and tournament series that need structured organiser roles, access control, and accountability across multiple events.

  • Planned
    Organiser accounts

    Dedicated organiser accounts with persistent access to event history, team data, and tournament configurations.

  • Planned
    Role-based access control

    Assign different permission levels to users — e.g. head organiser, scorekeeper, referee, read-only viewer — so each person sees and can do only what's appropriate for their role.

  • Planned
    Multi-organiser events

    Allow multiple people to co-manage a single event — useful for large tournaments with separate court managers or a dedicated administration team.

  • Planned
    Event-level permissions

    Control who can view, edit, or publish results for a specific event — independently of organisation-wide roles.

  • Planned
    Audit log

    A timestamped log of actions taken during an event — score changes, bracket edits, withdrawals — for dispute resolution and record-keeping.
