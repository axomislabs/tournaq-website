Quick Game

Quick Game focuses on providing quick access to one of TournaQs core elements: The Scoreboard — set up a game quickly and start scoring. You might set up a game between available or new teams and players, or save your time randomize a the settings and just start scoring.

Best for

  • Casual matches at the beach or training
  • Informal club sessions with fixed teams
  • Any situation where you just need a scorekeeper

Setting Up a Quick Game

Quick Start Game allows you to create a beach volleyball match quickly without any complex setup.

Select Match Format

Choose One Set or Best of Three Sets as the match format before entering team details.

Choose Teams

Select existing teams, create new teams manually, or let the app generate random teams.

Random Team Generator

Generate fun random team names, re-roll if needed, and start the match immediately.

Scoreboard

The Scoreboard is the main live scoring screen. It displays team names, scores, sets, player names, active player information and match actions.

Vertical Scoreboard

The vertical scoreboard provides a complete view for live scoring, including all information needed during a match.

Landscape Scoreboard

The landscape scoreboard is optimized for horizontal mobile use during live matches.

Match History

The Match History view shows a point-by-point log of the match, including who scored each point and when sides switched.

Match Controls

Match controls help you manage side changes, team positions, service order, and player names during live scoring.

Side Change Reminder

When the total score reaches the target, a dialog prompts teams to switch sides. Confirm to continue the match.

After Side Change

After confirming a side change, the teams are automatically swapped on the scoreboard so the display always reflects the physical court position.

Change Service

The active server is highlighted on the scoreboard. Service changes automatically when the non-serving team scores, and can also be adjusted manually.

Player Names & Service

Player names are shown directly on the scoreboard. They can be edited ad-hoc by tapping on the team name.

Set & Game Completion

Set Complete

When a team reaches the target score the set is completed. The result is locked and you can proceed to the next set or complete the game.

Game Complete

When the match is decided the game completion screen shows all set results and the winner. The match is saved to Match History.

Games & Match History

The Games screen shows in-progress and completed matches with team names, set scores, match status and the winner when the match is completed.

Step-by-Step Workflow

  1. Open TournaQ.
  2. Tap Quick Start Game.
  3. Select One Set or Best of Three Sets.
  4. Choose existing teams, create new teams or generate random teams.
  5. Start the game.
  6. Use the plus and minus buttons to score.
  7. Confirm side changes when prompted.
  8. Complete the set.
  9. Complete the game.
  10. Return to Games to review match history.
